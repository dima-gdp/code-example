# Примеры кода

Это примеры кода, которые я писал на работе.

## Структура проекта

### `/partner-programs-nuxt`

Страница настроек топа партнерских программ в социальной сети, написанная на Nuxt.

**Основные компоненты:**
- `components/operator/` - компоненты для управления партнерскими программами
  - `draggable/` - кнопки для drag-and-drop интерфейса
  - `form/` - формы управления
  - `program/` - компоненты отображения программ и топа
- `composables/` - композаблы для работы с партнерскими программами
- `pages/lk/partner-programs/settings.vue` - страница настроек
- `playwright/` - E2E тесты с использованием Playwright
  - `e2e/lk/auth/partner-programs/settings/` - тесты функционала
  - `__screenshots__/` - визуальные снапшоты для регрессионного тестирования

### `/ui`

Пример UI компонента с unit тестами.

**Содержит:**
- `Accordion.vue` - компонент аккордеона
- `accordion.test.ts` - unit тесты
- `accordion.stories.ts` - Storybook stories для визуальной документации
